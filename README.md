# Pizza Sales Analysis

## Overview

This project analyzes a pizza sales dataset to uncover meaningful business insights using data analytics and visualization techniques. By leveraging SQL for data processing and Power BI for dashboard creation, the project transforms raw transactional data into an interactive and insightful report.

The dashboard helps stakeholders monitor performance, identify trends, and make data-driven decisions to improve sales and operations.

## Objectives

* Analyze overall business performance using key KPI's
* Identify sales trends across days and months
* Understand customer preferences by pizza category and size
* Determine top-performing and underperforming products
* Provide actionable insights to improve revenue and efficiency

## Project Workflow

**Data Collection**
* Imported pizza sales dataset (CSV/Excel format)

**Data Cleaning & Preparation** 
* Removed null values and duplicates
* Standardized column formats (dates, categories, etc.)

**Data Analysis (SQL)**
* Calculated KPIs such as Revenue, Orders, Quantity
* Aggregated data by category, size, and time

**Data Modeling**
* Established relationships between tables (if applicable)
* Created calculated measures (DAX in Power BI)

**Data Visualization (Power BI)**
* Designed interactive dashboards
* Added filters (date range, category)

**Insights Generation**
*Identified trends, patterns, and business opportunities

## Tools & Technologies Used

* SQL – Data querying and KPI calculations
* Power BI – Data visualization and dashboard creation
* Microsoft Excel / CSV – Data source and preprocessing

## Dashboard Description

**Main Dashboard**

Displays key KPIs:
* Total Revenue
* Average Order Value
* Total Pizzas Sold
* Total Orders
* Average Pizzas per Order

**Visualizations include:**

* Daily trend of total orders (bar chart)
* Monthly trend of total orders (line chart)
* Sales distribution by pizza category (donut chart)
* Sales distribution by pizza size (donut chart)
* Total pizzas sold by category (bar chart)

**Best/Worst Sellers Dashboard**

**Highlights:**
* Top 5 pizzas by Revenue, Quantity, and Orders
* Bottom 5 pizzas by Revenue, Quantity, and Orders

**Helps identify:**
* Most popular pizzas
* Least performing items

## Key Insights
* Orders are highest on weekends, especially Friday and Saturday evenings
* January and July show peak sales performance
* Classic category contributes the most to total sales and orders
* Large-sized pizzas generate the highest revenue share
* Certain pizzas consistently dominate across revenue, quantity, and orders
* Some pizzas underperform across all metrics, indicating low demand

## Conclusion

The analysis provides a comprehensive view of pizza sales performance, highlighting key trends and customer preferences. The interactive dashboard simplifies complex data into actionable insights, enabling better decision-making for business growth.

## Recommendations

* Increase promotions on weekends to maximize already high demand
* Focus on best-selling pizzas by ensuring availability and marketing them more
* Re-evaluate underperforming pizzas (pricing, ingredients, or remove from menu)
* Promote large-size pizzas through combo deals or discounts
* Capitalize on peak months (January & July) with targeted campaigns
* Use insights for inventory planning and demand forecasting
